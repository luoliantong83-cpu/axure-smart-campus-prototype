# 智慧校园服务平台 - 交互说明文档

## 📖 文档说明
本文档详细说明了智慧校园服务平台各页面的交互设计、用户操作流程和功能实现细节。

---

## 1️⃣ 登录页面交互说明

### 页面元素
- **平台Logo** - 点击返回首页
- **学号/工号输入框** - 输入用户账号
- **密码输入框** - 输入用户密码
- **身份选择** - 下拉框选择（学生/教师/管理员）
- **记住密码复选框** - 勾选保存登录状态
- **登录按钮** - 提交登录表单
- **找回密码链接** - 跳转至密码重置页面
- **注册链接** - 跳转至注册页面

### 交互流程
```
1. 用户进入登录页面
2. 选择身份类型（学生/教师/管理员）
3. 输入学号/工号
4. 输入密码
5. 可选：勾选"记住密码"
6. 点击"登录"按钮
7. 系统验证身份信息
8. 验证成功 → 进入首页
9. 验证失败 → 显示错误提示
```

### 交互细节
- **输入验证**: 学号/工号和密码不能为空
- **错误提示**: 身份认证失败时显示红色错误消息
- **加载状态**: 登录按钮显示加载动画
- **记住密码**: 保存账号信息到本地存储
- **找回密码**: 跳转至邮箱验证页面

```

│  顶部导航栏（Logo + 用户菜单）      │

│├─ 社区│                            │
│└─ 个人│                            │
├──────┴────────┬──────────────────────────────────────┤
│      │  主内容区域                   │
│      │                              │
└──────┴──────────────────────────────────────────────┘
```

### 交互行为
1. **左侧菜单切换**
   - 点击菜单项切换对应页面
   - 当前页面菜单项高亮显示

   - 卡片悬停时显示阴影效果

   - 点击通知进入详情页面

4. **顶部用户菜单**
   - 点击用户头像显示下拉菜单
   - 菜单包含：个人中心、设置、登出

## 3️⃣ 课程管理交互说明

### 功能模块

#### 3.1 课表查询
- **周课表视图** - 显示当前周课程
- **周切换** - 上一周/下一周切换
- **课程点击** - 查看课程详细信息
- **打印功能** - 打印课表

#### 3.2 选课功能
```
选课列表页 → 搜索筛选 → 查看课程详情 → 选课确认 
→ 加入购物车 → 确认选择 → 成功提示 → 返回列表
```

**交互细节**:
- 搜索框支持课程名、教师名、课程号搜索
- 筛选条件：开课时间、学分、课程类型
- 课程冲突提醒
- 已选课程显示"已选"标签
- 人数已满课程显示"已满"标签

#### 3.3 课程详情
- 课程基本信息（名称、代码、学分）
- 教师信息（姓名、邮箱、办公室）
- 课程描述和教学大纲
- 选课人数和限选人数
- 上课时间和地点
- "加入购物车"或"确认选课"按钮

## 🎨 交互动画说明

### 页面转换动画
- **进入页面**: 从右侧滑入（200ms）
- **返回页面**: 向左滑出（200ms）
- **菜单展开/折叠**: 0.3s平滑过渡

### 按钮交互
- **悬停效果**: 背景色变暗，显示阴影
- **点击反馈**: 按钮压低效果
- **加载状态**: 按钮显示加载动画，文字变灰
- **禁用状态**: 按钮呈灰色，不可点击

### 列表交互
- **项目悬停**: 背景色变浅，显示右侧操作菜单
- **上拉加载**: 滚动到底部自动加载
- **下拉刷新**: 下拉显示刷新动画

## ⚠️ 特殊交互注意事项

1. **数据验证**
   - 所有输入框必须验证
   - 显示实时验证反馈

2. **错误处理**
   - 显示友好的错误提示
   - 提供解决建议

3. **确认操作**
   - 删除操作前显示确认弹窗
   - 重要操作需要二次确认

4. **加载状态**
   - 长操作显示进度条
   - 短操作显示加载动画

5. **反馈提示**
   - 操作成功显示绿色提示
   - 操作失败显示红色提示
   - 信息提示显示蓝色提示
