# 交互流程图

## 1. 用户认证流程

### 登录流程
```
┌──────────────────────────────────┐
│   输入账号密码   │
└──────────────────┬───────────────┘
         │
         ▼
┌──────────────────────────────────────────────┐
│  验证学号/工号是否正确   │
└──────────────────────┬──────────────────┬────┘
         │        │
    正确 │        │ 错误
         │        └──────▶ ❌ 显示错误提示
         │             返回输入
         │
         ▼
┌──────────────────────────────────────────────┐
│    验证密码是否正确      │
└──────────────────────┬──────────────────┬────┘
         │        │
    正确 │        │ 错误
         │        └──────▶ ❌ 显示错误提示
         │             返回输入
         │
         ▼
┌──────────────────────────────────────────────┐
│    验证身份选择          │
└──────────────────────┬──────────────────┬────┘
         │        │
    有效 │        │ 无效
         │        └──────▶ ❌ 要求重新选择
         │
         ▼
┌──────────────────────────────────────────────┐
│   登录成功，进入首页     │
└──────────────────────────────────────────────┘
```

## 3. 选课流程（详细）

```
┌──────────────────────────────────────────┐
│   进入选课页面    │
└──────────────────┬───────────────────────┘
         │
         ▼
┌──────────────────────────────────────────────┐
│   查看选课开放时间和说明      │
└──────────────────────┬───────────────────────┘
         │
         ▼
┌──────────────────────────────────────────────┐
│   浏览课程列表或搜索课程      │
└──────────────────────┴───┬──────────────┬────┘
          │        │
      找到课程  未找到
          │        │
          │        └──────▶ 修改搜索条件
          │             │
          │             ▼
          │         ┌──────────────────┐
          │         │ 重新搜索  │
          │         └──────────────────┘
          │              │
          └──────┬───────┘
                 ▼
┌──────────────────────────────────────────────┐
│   点击课程查看详细信息    │
└──────────────────────┬───┬──────────────┬────┘
          │        │
      满足条件 │        │ 不满足
          │        └──────▶ ⚠️ 显示警告
          │             返回列表
          │
          ▼
┌──────────────────────────────────────────────┐
│   点击"确认选课"或       │
│   "加入购物车"          │
└──────────────────────┬───┬──────────────┬────┘
          │        │
         │        └──────▶ 继续选择其他课程
         │
         ▼
┌──────────────────────────────────────────────┐
│   确认选课信息            │
└──────────────────────┬───┬──────────────┬────┘
          │        │
      确认 │        │ 取消
         │        └──────▶ 返回列表
         │
         ▼
┌──────────────────────────────────────────────┐
│   选课成功，进入课表页    │
└──────────────────────────────────────────────┘
```

## 8. 模态框和弹窗交互

### 确认弹窗
```
用户操作 → 触发确认弹窗
         │
    ┌────┴────┐
    │      │
  确认      取消
    │      │
    ▼      ▼
执行操作  关闭弹窗
        返回
```

### 错误提示
```
操作失败 → 显示错误消息
         │
       确认
         │
      关闭提示
      用户可重试
```

### 成功提示
```
操作成功 → 显示成功消息
         │ (自动消失或确认)
      关闭提示
      继续操作
```

## 总结

上述交互流程涵盖了平台的主要功能模块，清晰地展示了用户从进入某个功能到完成操作的全过程。每个流程都考虑了不同的用户操作结果和相应的处理方式，确保用户体验的连贯性和友好性。
