# 智慧校园服务平台原型设计

## 项目概述

本项目是一个完整的智慧校园服务平台的Axure原型设计，包含学生、教师和管理员等多个用户角色的功能模块。

## 🎯 主要功能模块

### 1. 用户认证系统
- **登录界面** - 学生/教师/管理员三种身份登录
- **账户注册** - 新用户注册流程
- **找回密码** - 密码重置功能
- **身份选择** - 多身份切换

### 2. 首页/仪表板
- **快速导航** - 常用功能快捷入口
- **通知提醒** - 实时校园通知展示
- **个人卡片** - 用户基本信息展示
- **功能概览** - 各模块功能展示

### 3. 📚 课程管理
- **课表查询** - 周课表、学期课表查询
- **选课功能** - 课程选择和取消
- **课程详情** - 课程信息、教师信息、课程描述
- **教学日历** - 校历、考试时间安排
- **课程评价** - 学生对课程的评价反馈

### 4. 📊 成绩查询
- **学期成绩** - 各学期成绩汇总
- **课程成绩详情** - 单门课程详细成绩
- **成绩统计** - GPA、排名统计
- **成绩打印** - 成绩单导出打印

### 5. 📖 图书借阅系统
- **图书搜索** - 按书名、作者、ISBN搜索
- **图书详情** - 图书信息、库存、评价
- **借阅记录** - 已借、已还、超期提醒
- **续借功能** - 在线续借申请
- **预约服务** - 预约图书功能

### 6. 📢 校园通知
- **通知列表** - 学校、学院、专业通知分类
- **通知详情** - 查看完整通知内容
- **通知分类** - 学术、生活、活动等分类浏览
- **通知订阅** - 自定义通知接收设置

### 7. 💬 校园社区
- **社区首页** - 热门话题、推荐帖子
- **话题浏览** - 分类浏览社区话题
- **发布帖子** - 创建新话题
- **评论互动** - 回复、点赞、收藏
- **用户主页** - 查看其他用户资料

### 8. 👤 个人中心
- **个人信息** - 学号、姓名、专业、班级等
- **账户设置** - 修改密码、绑定邮箱手机
- **隐私设置** - 隐私信息管理
- **消息中心** - 私信、系统通知、提醒
- **我的收藏** - 收藏的帖子、资源等

## 📱 支持平台

- 📱 **移动端** - iPhone / Android响应式设计
- 💻 **桌面端** - Windows / Mac / Linux
- 🖥️ **平板端** - iPad等平板设备

## 🔄 交互流程

### 用户登录流程
```
登录页面 → 输入学号/工号 → 输入密码 → 选择身份 
→ 验证身份 → 进入首页 → 系统首页
```

### 选课流程
```
首页 → 课程管理 → 选课列表 → 选择课程 
→ 确认选课 → 成功提示 → 返回课表
```

### 图书借阅流程
```
首页 → 图书借阅 → 搜索图书 → 查看详情 
→ 确认借阅 → 成功借阅 → 查看借阅记录
```

## 📋 使用说明

### 打开Axure文件
1. 下载 `smart-campus-platform.rp` 文件
2. 用Axure RP 8.0或更高版本打开
3. 在Axure中编辑、预览、发布原型

### 预览原型
1. 在Axure中点击 **Preview** 预览
2. 或生成 HTML 文件进行在线预览
3. 可在不同设备上进行响应式测试

### 导出说明
- 可导出为HTML、PDF、Word等格式
- 截图位于 `screenshots/` 文件夹
- 完整文档位于 `docs/` 文件夹
